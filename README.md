     7	## 📖 プロジェクト概要
     8	
     9	**SAMPA**は、ブラジル・サンパウロ在住の日本人および日系人向けの生活情報ウェブマガジンです。カナダ・トロントで人気の「Torja」をモデルに、サンパウロの生活情報、グルメ、ビジネス、カルチャーなど、在住者に役立つ情報を日本語で提供します。
    10	
    11	### 🎯 ターゲットユーザー
    12	- サンパウロ在住の日本人駐在員
    13	- ブラジル在住の日系人
    14	- ワーキングホリデー・留学生
    15	- サンパウロへの移住を検討している方
    16	- ブラジルとの仕事・ビジネスをしている方
    17

    40	- フッター上広告（全幅バナー）

    43	- 6名のコラムニスト紹介

    62

   161	## 🚀 使い方
   162	
   163	### ローカル環境での実行
   164	
   165	1. **ファイルのダウンロード**
   166	   ```bash
   167	   # プロジェクトフォルダに移動
   168	   cd sampa-website
   169	   ```
   170	
   171	2. **ブラウザで開く**
   172	   - `index.html` をダブルクリック
   173	   - または、ブラウザのアドレスバーにファイルパスを入力
   174	
   175	3. **ライブサーバーでの実行（推奨）**
   176	   ```bash
   177	   # VS Codeの場合
   178	   # Live Server拡張機能を使用
   179	   # index.htmlを右クリック → "Open with Live Server"
   180	   ```
   181	
   182	### 本番デプロイ
   183	
   184	#### 静的ホスティングサービス
   185	- **Netlify**: ドラッグ＆ドロップでデプロイ
   186	- **Vercel**: GitHubリポジトリ連携
   187	- **GitHub Pages**: 無料ホスティング
   188	- **AWS S3**: S3バケット + CloudFront
   189	
   190	#### デプロイ手順（Netlify例）
   191	```bash
   192	1. https://www.netlify.com/ にアクセス
   193	2. "Add new site" → "Deploy manually"
   194	3. index.htmlをドラッグ＆ドロップ
   195	4. デプロイ完了（自動でURLが発行される）
   196	```
